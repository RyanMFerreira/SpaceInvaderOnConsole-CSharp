# Space Invaders On Console - C#

Este projeto tem como objetivo desenvolver uma versão do clássico jogo **Space Invaders** para ser executado no console, utilizando a linguagem **C#**. 

## Informações do Projeto

### Objetivo
O objetivo principal deste projeto é recriar a experiência do jogo *Space Invaders* de forma simplificada, utilizando apenas texto e caracteres ASCII para representar os elementos gráficos.

### Tecnologias Utilizadas
- **Linguagem:** C#
- **Plataforma:** Console

### Funcionalidades Implementadas
- **Menu Básico:** Implementação de uma interface intuitiva.
- **Renderização do conteúdo:** Renderiza os elementos no Console.
- **Movimentação do jogador:** Permite que o jogador se mova e dispare projéteis para destruir os inimigos.
- **Colisões:** Detecta colisões entre o jogador e as extremidades do mapa.

### Próximos Passos
1. Adicionar inimigos:
   - Design dos inimigos;
   - Comportamento e padrões de movimento.
   - Inimigos devem ser capazes de se mover e disparar contra o jogador.
2. Implementar níveis de dificuldade:
   - Diferentes níveis que aumentam a dificuldade progressivamente.
3. Sistema de pontuação:
   - Desenvolver um sistema para calcular e exibir a pontuação do jogador.
4. Melhorar a interface do usuário:
   - Criar uma interface mais intuitiva e visualmente agradável.

#### Requisito Opcional:
- Implementação das artes ASCII nas naves e inimigos;
- Escolha entre duas possíveis naves diferentes;
